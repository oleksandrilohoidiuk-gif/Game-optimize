AI Game Optimizer

Веб-застосунок на основі Node.js, який використовує штучний інтелект (LLM) для генерації персоналізованих налаштувань графіки у відеоіграх. Застосунок аналізує ваше "залізо", обрану гру та пріоритет (FPS чи Якість), і видає оптимальні параметри для комфортної гри.

Основні можливості

ШІ-асистент: використовує Groq API (модель Llama 3.1) для генерації точних технічних порад.
Також у вас є можливість обрати пріоритет оптимізації (Максимальна продуктивність, Найкраща якість або Баланс).
Історія запитів: усі згенеровані налаштування автоматично зберігаються у хмарну базу даних.

Використані технології

Backend: Node.js, Express.js
Frontend: HTML/CSS + Handlebars (`.hbs`)
База даних: PostgreSQL (Neon Tech)
AI Integration: Groq API 

Запуск

npm install
npm install express hbs pg dotenv
